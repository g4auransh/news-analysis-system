Intelligent News Summarization and Sentiment Analysis System
A comprehensive NLP tool designed to summarize news articles, perform sentiment analysis, extract key entities, and answer questions using Large Language Models.

Features
Summarization: Condenses long-form articles into concise summaries.

Sentiment Analysis: Identifies the emotional tone (Positive/Negative) of the news.

Entity Extraction: Highlights key people, organizations, and locations.

Interactive UI: A web-based interface built with Streamlit for easy interaction.

Prerequisites
Python 3.10+

pip (Python package manager)

Installation
Clone the repository:

Bash
git clone https://github.com/yourusername/news-analysis-system.git
cd news-analysis-system
Create a virtual environment:

Bash
python -m venv venv
# On Windows:
venv\Scripts\activate
# On Ubuntu/Linux:
source venv/bin/activate
Install dependencies:

Bash
pip install -r requirements.txt
Running the Application
Launch the web interface:

Bash
streamlit run app.py
Access the app: Open the local URL provided in the terminal (usually http://localhost:8501) in your browser.
